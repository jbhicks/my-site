# My Web Site
Objective: Create a highly performant personal web site for hosting personal projects that makes use of modern web frameworks for UI while also utilizing new technologies that allow for shipping much less Javascript to the browser.

## Tech Stack
I know I want to investigate some sort of static site generator tool in the pursuit of the first goal of the project - speed. After evaluating both [Next.js](https://nextjs.org/) and [Astro](https://astro.js.org/), I landed on Astro due to the tight integration with Markdown files. For making standalone web components, [Lit](https://lit.dev/) will be used to help keep the Javascript that is required to an absolute minimum.

## Template
Big thanks to Maxi Ferreira for the template and features section with the excellent [Astro blog template](https://github.com/Charca/astro-blog-template).

## ✨ Features:
- ✅ Astro 1.0
- ✅ Dark Mode
- ✅ Full Markdown support
- ✅ SEO-friendly setup with canonical URLs and OpenGraph data
- ✅ RSS 2.0 generation
- ✅ Sitemap.xml generation
